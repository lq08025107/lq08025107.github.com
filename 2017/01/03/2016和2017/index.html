2016和2017

2016年过去了，今天是2017年第一个工作日，2017年1月3日。

新年是回家过得，跟爸妈姐姐很开心，体会到家人身体健康开心快乐才是最重要的。谢谢你们。

以年度总结的角度去想2016，想到哪里写到哪里：

  1. 前半年纯属浪费时光，没有收获，却失去了奶奶。
  2. 后半年从0开始走入后端，逐渐入门。
  3. 目前了解主流技术路线，了解互联网后端的大体架构。
  4. 内功不足，基础薄弱，没有对一些框架进行思考和深入研究，仅仅停留在会用阶段。

2017年，期望：

  1. 加强内功修炼，着重Java和JS
  2. 跟紧潮流，熟悉新的框架
  3. 根据自己脑中idea，慢慢着手自己动手写框架
  4. 有机会的话，熟悉安卓。

2016和2017关键字：

奶奶 姐姐宝宝 RPC dubbox REST docker swarm netty Elasticsearch 微服务 Spring spring boot
